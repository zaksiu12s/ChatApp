## SETUP
1. Download all code
```bash
git clone https://github.com/zaksiu12s/ChatApp.git
cd ChatApp
 ```
2. cd into client and into api:
   - in client run:
```bash
npm i
npm run start
```
  - in api run:
```bash
npm i
npm run build
npm run start
```
